# 🛍️ 商城访问指南

## 快速访问

### 方式1：直接访问URL（最简单）

1. **启动前端**
   - 双击运行 `START_FRONTEND.bat`
   - 或在终端执行：`npm run dev`

2. **访问商城首页**
   ```
   http://localhost:8888/#/shop
   ```

3. **访问推广链接**（测试佣金计算）
   ```
   http://localhost:8888/#/shop/product/1?distributorId=1
   ```

## 📋 商城页面导航

### 公开页面（无需登录）

| 页面 | URL | 说明 |
|------|-----|------|

## 🧪 测试流程

### 场景1：普通购物流程

1. 访问商城首页：`http://localhost:8888/#/shop`
2. 浏览商品列表
3. 点击商品查看详情
4. 点击"加入购物车"
5. 点击顶部购物车图标
6. 选择商品，点击"去结算"
7. 选择收货地址，提交订单
8. 模拟支付

### 场景2：推广订单流程（测试佣金）⭐

1. 访问推广链接：`http://localhost:8888/#/shop/product/1?distributorId=1`
2. 查看商品详情（应显示"由分销商推荐"）
3. 点击"立即购买"
4. 选择收货地址，提交订单
5. 模拟支付
6. **验证佣金计算**：
   - 查看后端日志
   - 查询数据库 `order_commission` 表
